Alzheimer's Disease Analysis Software (ADIPOS)

Welcome to Our Hospital Software

This software is designed to assist doctors and administrators in predicting Alzheimer's disease using topological data analysis.

With our user-friendly interface, medical professionals can input patient data and receive predictions and insights to aid in diagnosis and treatment planning.

Administrators can manage user accounts, access control, and view system analytics to ensure smooth operation of the software.
